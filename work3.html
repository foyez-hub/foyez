003 : Stock Data Visualization Web Application

GitHub Link Video Demo

Web app for visualizing stock data with CRUD operations, line and bar chart visualization, and deployment on Netlify and PythonAnywhere.

Tools & technologies used: ReactJS, Django, JavaScript, Python, SQL (SQLite3), HTML, CSS

  • Table Visualization: Displays stock data with editable rows and pagination.
  • CRUD Operations: Enables Create, Read, Update, Delete operations on stock data.li>
  • Line and Bar Chart Visualization: Visualizes 'close' and 'volume' data with multi-axis chart and filtering by trade_code.li>
  • Additional Data Visualization: Dropdown menu for viewing 'High', 'Low', 'Open' columns in charts.
